Projects

Conflict Graph Node Removal Simulator

This project simulates the removal of nodes from a conflict graph and measures which removal algorithm is most effective at balancing utility vs. fairness. I implemented the simulation using python and the networkx library.

Dynamic Memory Allocator

This project implements a dynamic memory allocator in C. The allocator supports malloc, free, and realloc operations, and uses a best fit strategy for allocating memory. I implemented the allocator using a linked list to manage free blocks of memory and a bitmap to track allocated blocks.
